# Color

Hony UI 为了避免视觉传达差异，使用一套特定的调色板来规定颜色，为你所搭建的产品提供一致的外观视觉感受。

## 品牌色

它常常作为品牌的标志性色调，旨在塑造整体的视觉形象或强调关键元素。

<ClientOnly>
<BrandColorPalette />
</ClientOnly>

## 功能色

明确的信息以及状态，比如成功、出错、失败、提醒、链接等

</ClientOnly>

若图标与文字共同呈现时，图标颜色的调整应**与文字颜色保持一致性**。
